# MyRules Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the MyRules v1 skill + sync toolchain that deploys platform-neutral rules from `~/.myrules/` to Cursor and Claude Code, with opt-in legacy prune, export, and gitignored deploy artifacts.

**Architecture:** Single GitHub repo holds `rules/`, `manifest.yaml`, `skills-manifest.yaml`, and `tools/sync/` scripts. Each machine clones to `~/.myrules/`. Consumer projects install `skills/myrules/SKILL.md` only; sync transforms Markdown sources into `.cursor/rules/myrules-*.mdc` and `.claude/rules/myrules-*.md` (plus Claude user dir). Bash scripts are the reference implementation; PowerShell mirrors behavior on Windows.

**Tech Stack:** Git, Bash, PowerShell 5+, YAML (parsed minimally in shell or via small inline logic), JSON for sync state. No Node/Python runtime required for v1.

## Global Constraints

- v1 platforms: **Cursor + Claude only** (no Codex)
- Managed prefix: **`myrules-`**
- Cursor user rules via **per-project** `.cursor/rules/myrules-user-*.mdc` with **`alwaysApply: true`** (not Settings UI)
- Claude project rules: **`.claude/rules/myrules-*.md`** without `paths` frontmatter
- Protect never touch: **`CLAUDE.md`**, **`.claude/CLAUDE.md`**, **`CLAUDE.local.md`**, **`AGENTS.md`**, **`~/.claude/projects/**/memory/**`**
- Deploy artifacts **gitignored by default** (`deploy.gitignore_deploy_artifacts: true`)
- Prune requires prior dry-run: **`pruneDryRunDone: true`** in `.myrules-sync-state.json`
- Prune targets include: **`.cursorrules`**, **`.cursor/rules/imported/**`**, non-`myrules-*` rules
- **`skills-manifest.yaml` lists external skills only** — never MyRules itself
- Cache path: **`~/.myrules/`**

Deploy logic:

```bash
myrules_deploy_rules() {
  local cache="$1" project="$2"
  local cursor_dir claude_proj_dir claude_user_dir
  cursor_dir="$(myrules_get_cursor_rules_dir "$project")"
  claude_proj_dir="$(myrules_get_claude_project_rules_dir "$project")"
  claude_user_dir="$(myrules_get_claude_user_rules_dir)"
  mkdir -p "$cursor_dir" "$claude_proj_dir" "$claude_user_dir"

  for f in "$cache/rules/user/"*.md; do
    [[ -f "$f" ]] || continue
    topic="$(basename "$f" .md)"
    myrules_transform_cursor "$f" "$topic" true > "$cursor_dir/myrules-user-${topic}.mdc"
    myrules_transform_claude "$f" > "$claude_user_dir/myrules-${topic}.md"
  done

  for f in "$cache/rules/project/"*.md; do
    [[ -f "$f" ]] || continue
    topic="$(basename "$f" .md)"
    myrules_transform_cursor "$f" "$topic" false > "$cursor_dir/myrules-${topic}.mdc"
    myrules_transform_claude "$f" > "$claude_proj_dir/myrules-${topic}.md"
  done
}
```
